query = {"设整数 \( n \geq 2 \)。定义集合
\[
S_n = \left\{ (x_1, x_2, \dots, x_n) \in \mathbb{Z}^n \,\middle|\, 1 \leq x_1 < x_2 < \dots < x_n \leq 2n,\ x_1 + x_2 + \dots + x_n \equiv 0 \pmod{2n+1} \right\}.
